# 斯特拉福德国际大学品牌指南

## 校徽使用规范

### 官方校徽

斯特拉福德国际大学的官方校徽是一个蓝金色圆形标志，包含"SIU"字样和学校全名。校徽应在所有官方文件、证书、网站和宣传材料中使用。

### 颜色规范

- **深蓝色**: #003087 (RGB: 0, 48, 135)
- **金色**: #D4A017 (RGB: 212, 160, 23)
- **白色**: #FFFFFF (RGB: 255, 255, 255)

### 尺寸规范

- **最小尺寸**: 校徽不应小于20mm或60像素
- **比例**: 校徽必须保持1:1的宽高比
- **留白**: 校徽周围应保留至少校徽宽度10%的留白空间

### 使用场景

1. **证书**: 校徽应放置在证书顶部中央位置，尺寸为60mm
2. **网站**: 校徽应放置在网站页眉，尺寸为40px
3. **印章**: 校徽应作为官方印章的核心元素
4. **信头**: 校徽应放置在信头左上角，尺寸为25mm

### 禁止事项

- 不得更改校徽颜色
- 不得扭曲或变形校徽
- 不得在低对比度背景上使用校徽
- 不得添加额外的装饰元素到校徽

## 文件格式

校徽提供以下格式:

1. **SVG**: 用于网站和数字媒体
2. **PNG**: 用于需要透明背景的场景
3. **JPG**: 用于印刷材料

所有格式的文件都可在 `/public/images/siu-logo/` 目录中找到。
